Final assignment WS 2019/20

The assignment is to carry out a spatio-temporal analysis on a dataset of your choice, using one of the methods that we dealt with during the course so far, or another (after approval), and report on this.

The report shall:

  • deal with a problem that is solved by analysing spatiotemporal data (one or more of: spatial time series, geostatistical data, lattice data, point patterns, satellite imagery, movement data)
  • have 1000-2000 words of free text (i.e., excluding the analysis code)
  • be structured like a short scientific report (have the following sections: introduction, data, methods, results, discussion and conclusion, list of references)
  • state in the introduction clearly the research question (or hypothesis)
  • the reference list shall include at least three pieces of related work
  • each of the works listed in the references shall be referred to in the main text
  • be written in scientific style
  • be kept simple
  • the analysis shall be part of the text flow, figures (and/or tables) shall support the discussion and conclusions, and shall not be suppressed on output

In case you decide to build upon an existing R study (e.g., a vignette or demo script in an R package), then:

  • your assignment shall properly refer to the work you build upon
  • your assignment shall include reading the data from a file (that is: not using R's data() statement; this may mean you export the dataset once and import it in your assigment work).
  • your assignment shall go beyond the analysis of the R script you based upon, and not do an identical analysis.

The report must be submitted as a single .zip file with

  • the report in an R-Markdown file (or similar),
  • the data used,
  • the final, readable form (either html or pdf)

The report shall be reproducible based on the included .Rmd file and the material in the .zip file.
