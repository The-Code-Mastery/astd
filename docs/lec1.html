Goals of this course

For the participants to develop

  • an idea about which types of spatio-temporal data there are, what kind of questions we can ask to these data, and how these data can be analysed
  • hands-on experience with some of these analysis methods
  • experience with common difficulties found in real world datasets
  • software skills for analysing spatio-temporal data
  • skills to make analyses reproducible, and communicate these

Some general advice:

  • always work with the latest released verion of R and Rstudio, and of packages
  • if on Mac OSX or Windows, use binary packages unless you know what a compiler is
  • regularly update your packages with update.packages()
  • if installation fails, try to remove a package first with remove.package()
  • never ignore warnings: they are there for a purpose

Review/refresher questions:

  1. How is the mean for a random variable \(X\) defined?
  2. Give the equation for covariance between two variables \(x\) and \(y\)
  3. Give the equation for correlation between two variables \(x\) and \(y\)
  4. For the samples \(x\) and \(y\), compute the sample mean, the variance and the standard deviation
  5. Draw the data in a scatter plot (by hand)
  6. From the table above, compute the covariance between \(x\) and \(y\), and the correlation between \(x\) and \(y\)
  7. Suppose \(x\) is a time series data, and the rows indicating the time index. Compute the lag 0, lag 1 and lag 2 autocorrelations of \(x\).
  8. What is the general equation of an autoregressive AR(p) process?
  9. What is the general equation of a moving average MA(q) process?
  10. Up to which lag is an AR(p) process correlated?
  11. Up to which lag is a MA(q) process correlated?
  12. What can be said about partial correlations of an AR(2) process?
